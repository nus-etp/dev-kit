# Screenshots

Drop release screenshots here and reference them from `../highlights.html`.

Convention (matches `artifacts/v_0_2_0/screenshots/` in etp-compass):

- Name files `NN-short-name.png` — the `NN` prefix orders them, e.g.
  `01-sign-in.png`, `02-admin-dashboard.png`.
- One screenshot per feature section in `highlights.html`; reference as
  `<img src="screenshots/NN-short-name.png" alt="…" />`.
- PNG, full app-window width — the `.shot` CSS scales them to the card width.

This file is a placeholder so the empty directory is tracked in git; delete it
once you add real screenshots.
